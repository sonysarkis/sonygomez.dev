# Sony Gómez — High-End Interactive Portfolio (2026)

A premium, high-performance developer portfolio built with a focus on immersive user experience, advanced animations, and modern web aesthetics.

## 🚀 Key Features
- **ParticleWeb Hero**: Interactive particle background with localized mouse coordinate mapping and dynamic resizing.
- **VapourText**: Custom text effect that dissolves typography into smoke particles during transitions.
- **CursorGlowTrail**: A fluid, liquid-like glowing trail that follows the cursor using HTML5 Canvas.
- **Spotlight Cards**: Premium UI cards with interactive radial gradients and glassmorphism.
- **Auto-Scroll Previews**: Project cards that automatically reveal the full page layout on hover.

## 🛠️ Tech Stack
- **Frontend**: React 19 + Vite
- **Styling**: Tailwind CSS v4 (Vanilla CSS architecture)
- **Animations**: Framer Motion
- **Graphics**: HTML5 Canvas API (Particle physics & effects)
- **Interactivity**: Custom ResizeObservers and Hook-based event management

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run in development mode**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```
